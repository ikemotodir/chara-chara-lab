FILE 01｜主人公・おばけ

パッチ Patti the Spook

二次創作OK
  • すがた：1つ目。目の下にクマ。ひまわり色のリボン
  • プロフィール：おばけの子ども・男の子。身長約60cm、体重約3kg
  • 性格：普段は仏頂面。でも楽しいことが大好きで、ほんとうは表情豊か
  • リボン：おじいちゃんからもらった宝物。取られると激怒する
  • 能力：いろんなものに変身できる。でも1つ目だけは残る。空も飛べる（長時間は無理）
  • 苦手：バナナ
  • ひみつ：毎日「おどろかせかた」を勉強してる。でも実は——ニンゲンが大好き
  • 友達：おばけ学校の同級生、ミーンとモゥ
FILE 02｜ヒロイン・ニンゲン

マユちゃん Mayu

公式作品のみ
  • プロフィール：おばけの街に迷い込んだ、ニンゲンの女の子
  • パッチとの関係：マユちゃんと出会って、パッチは「ニンゲンと仲良くなりたい」と思うようになった
🔒 二次創作の対象外です

マユちゃんをはじめとする公式の人間キャラクターは、公式作品だけの存在です。かわりに、あなたのオリジナル人間キャラクターをおばけの世界に迷い込ませてあげてください。

FILE 03｜世界観

おばけの街「No Life Town」 ノーライフタウン

二次創作OK
  • 場所：ニンゲンがすむ世界の、すぐとなり
  • 住人：おばけたちはニンゲンをビックリさせるのが大好き
  • 文化：ニンゲンの世界のものを、おばけ風に解釈して街に取り入れている
  • ルール：この街には「お金」という概念がない。やりたいことをやって、だれかのためになるおばけが豊かになる
  • テーマ：周りとは違う挑戦をすることで、夢はかなう

ミーン、モゥをはじめとする他のおばけたちの設定資料も、順番に追加していきます。
